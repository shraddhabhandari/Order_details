# Order_details
Data Analytics Project to Analyze Customer Order Summary Data
End-to-End Data Analytics Project (Python → SQL Server → Power BI)

This project demonstrates a complete end-to-end data analytics workflow, starting from data loading and preprocessing using Python (Pandas), performing structured data analysis in SQL Server, and finally building an interactive Power BI dashboard to visualize key business insights.

The primary objective of this project is to analyze customer order data to generate actionable insights related to:
1. Revenue performance
2. Profit analysis
3. Regional sales distribution
4. Channel performance
5. Order status tracking

Project workflow:
1. Data preparation and loading (w/ Python Pandas)
2. Data analysis in SQL Server
3. Power BI Dashboard
4. Finding insights and recommendations
